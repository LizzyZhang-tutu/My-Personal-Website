Missile Defense

This is a dodging survival game in which you control the warplane to turn right or left while flying to avoid missiles and collect gems.

Your goal is to collect as many gems as you can and flying as longer as you can while protecting yourself from hunting missiles, there are borders that you can not touch as well. The time in seconds that you remain flying will be counted 1 point each second, and the gems you collected will be counted for 10 points each. By accumulating points, you can purchase higher versions of warplanes (they are faster or/and more agile while turning), there are four warplane models in the shop that you can buy and try!

Here is a brief summary of what I did:

🚄 Designed UI for starting, gaming and gameover scene

🚄 Programmed to randomly generate missiles to trace warplane

🚄 Scripted XML files to store warplanes' data, built purchasing function by reading and modifying XML files

🚄 Strengthened object-oriented programming skills using API regarding vectors and quaternion

Monkey Cube Run

This is a parkour survival game in which you control the cube to go forward on tiles by tapping the screen, to survive you will need to avoid holes, ground-spikes and smashing-spikes😱. You will also find rewards to pick up on your way surviving!

Your goal is to go as far as you can, each step(tap) you take count for 1 point and each award you get is also counted. Be aware that there will be more and more obstacles appearing as time goes.

Here is a brief summary of what I did:

🚄 Devised ground pieces using Array and List data structure

🚄 Generated several types of obstacles and rewards randomly

🚄 Stored scores and reward quantity using PlayerPrefs

🚄 Established tapping control for the running cube by using related API

🚄 Managed several classes' scripts

🚄 Packaged the mobile game on Android Platform using JDK and SDK
